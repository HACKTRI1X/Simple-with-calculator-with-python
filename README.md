# 🧮 Basit Hesap Makinesi (Python)

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)

Python ile yazılmış, terminal üzerinden çalışan basit ve kullanışlı bir hesap makinesi uygulaması.

## ✨ Özellikler
- Toplama, Çıkarma, Çarpma, Bölme işlemleri
- Ondalıklı sayı (`float`) desteği
- Sıfıra bölünme (`ZeroDivisionError`) hatası yönetimi
- Hatalı giriş (`ValueError`) kontrolü
- Döngüsel yapı (istediğiniz kadar işlem yapabilirsiniz)

## 🚀 Kurulum ve Kullanım

Kodu bilgisayarınıza indirin ve Python yüklü bir sistemde çalıştırın:

```bash
python calculator.py
```
